# Paste App

A lightweight pastebin-style app to create, edit, search, and manage text snippets (“pastes”) in the browser.

## Highlights

- Create / update pastes (title + content)
- Search pastes by title
- View, copy, share, and delete pastes
- Uses `localStorage` for persistence (no backend)

## Run Locally

The React app is inside `paste-app/`.

```bash
cd paste-app
npm install
npm run dev
```

## Tech Stack

React (Vite), Redux Toolkit, React Router, Tailwind CSS, React Hot Toast.
